Jasmine-Closure-Sublime

Sublime 2 plugin to generate Jasmine test stubs

To install:

Clone the repository into your sublime-text-2/Packages directory. The package will add the command Jasmine: Open Jasmine Spec. When run from a JavaScript file, this command will search for the corresponding "spec" file; if this file does not exist, it will open a new file at that location and fill that file with a basic spec template.
